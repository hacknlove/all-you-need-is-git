## Install script

```bash
curl -fsSL https://aynig.org/install.sh | bash
```

To install a specific version:

```bash
VERSION=v0.1.0 curl -fsSL https://aynig.org/install.sh | bash
```

If your `PATH` does not include `~/.local/bin`, add it to your shell profile.
